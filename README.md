# 🎓 Student Performance Analytics System

## 📌 Project Description
This project is a Java-based application designed to analyze student performance using object-oriented programming and data structures.

It allows users to input student details and marks, then calculates performance metrics such as average score, grade, weakest subject, and provides smart suggestions for improvement.

---

## 🎯 Features
- Add student details (roll number and name)
- Enter marks for 5 subjects
- Calculate average marks
- Assign grades (A, B, C, Fail)
- Identify weakest subject
- Provide performance-based suggestions
- Sort students based on performance (ranking system)
- Display all student records

---

## 🛠️ Technologies Used
- Java
- OOP Concepts (Classes & Objects)
- ArrayList (Dynamic Data Storage)
- Control Structures (Loops, Conditionals)
- Comparator (for sorting)
